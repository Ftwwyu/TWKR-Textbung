## 이게 뭡니까?

Eratoho TWKR 0.15.0-beta 2를 기반으로 작성된 사가판(私家版) 게임입니다

보통의 TWKR에선 즐거운 미연시를 즐길수 있지만 그런 모습은 현실의 TextbungE와 맞지 않습니다
따라서 난이도를 상향, 기능을 추가및 수정한 버젼입니다


물론 추가된건 별로 없으니 그리 기대는 마시고...
